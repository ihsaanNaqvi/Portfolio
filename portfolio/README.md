# Syed Ihsaan Abbas Naqvi — Portfolio

Personal portfolio website. A single React site with three audience modes — **Engineering**, **Data Science**, and **Research / PhD** — that re-frame the same body of work for each visitor.

## Tech
- React 18 + Vite
- No external UI libraries (single-file component, inline styles)
- Fully responsive, accessible, respects reduced-motion

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown (usually http://localhost:5173).

## Build for production

```bash
npm run build
```

Output goes to `dist/`.

## Deploy

### Vercel (recommended)
1. Push this repo to GitHub.
2. Go to vercel.com → Add New Project → import this repo.
3. Framework preset: **Vite**. Click **Deploy**.
4. Live in ~1 minute at `your-project.vercel.app`.

### Netlify
1. Push to GitHub.
2. netlify.com → Add new site → import repo.
3. Build command: `npm run build` · Publish directory: `dist`.

### GitHub Pages
Use the `dist/` output with the `gh-pages` package, or prefer Vercel/Netlify for simplicity.
